# 库间调拨

---

## 新建调拨单

把存货从一个仓库转移到另外一个仓库，这就是库间调拨业务的基本逻辑。

通过新建调拨单，可以录入调拨信息，开始调拨业务流程。

## 编辑调拨单

调拨单在提交之前，可以编辑修改里面的单据数据。

## 查看调拨单

调拨单在提交之后，就不能修改数据，但是可以查看调拨单的数据。

## 删除调拨单

调拨单在提交之前，可以从系统中删除该调拨单。

## 提交调拨单

提交调拨单后，调拨业务完成，根据调拨单的数据实时更改库存账里面的存货数据。

## 单据生成PDF文件

调拨单生成PDF文件，可以供打印单据使用。

## 打印
